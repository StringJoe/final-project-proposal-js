Final Project: Turtles VS Dragons

PROJECT REQUIREMENTS:

  1. Product Name: Turtles VS Dragons
  2. Team Members: Joseph Santay
  3. Application Overview: In this game the goal of the user/player is to kill all dragons before they get past the edge of the screen. The player will have a limited amount of lives that replenish after each wave is finished. As the game progresses the waves will continue to get harder until it is impossible to continue or the user decides to quit. There will be different kinds of turtles that the player can use to defend their base. They each have their own individual magic abilities and weapons they can use to defend their base from attack.
  4. Technology Overview: From a technical standpoint, this game will simply use the built in canvas tag in HTML to accomplish its goals. While it is possible a library will be used in the creation of the game, there is no current plan to use one right now. This website will be hosted on Microsoft Azure.
  5. Proposal Last Updated: 11/20/2022
  6. Feature List:
    • Turtle Sprites
    • Dragon Sprites
    • Player Lives
    • Player Movement
    • Enemy Waves
    • Player Attacks
    • Map
    • Top 10 Leaderboard
  7. Future Feature List:
    • Pause Button
    • Time Limit
    • Multiple Maps
    • Muliple Players
    • Time Limit
  8. Basic Technical Features: This Website will use some basic HTML and CSS to manipulate the webpage. There will also be some basic JavaScript used to implement some of the player info.
  9. Advanced Technical Features: The only advanced technical feature being used for this project is the canvas tag and the JavaScript used to manipulate it. Sprite sheets will also be used to implement the characters in the game. While images have been used in this class, the sprite sheets will have multiple character drawings on it making it more complex
